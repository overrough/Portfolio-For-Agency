# Elevate Web Works — Premium 3D Portfolio

A high-performance, 3D-enhanced digital agency portfolio built with Next.js 14, React Three Fiber, Framer Motion, and Tailwind CSS.

## 🚀 Features
- **Dynamic 3D Hero:** Immersive WebGL-powered icosahedron with glass transmission materials.
- **Conversion Psychology:** Built for high-impact landing page performance.
- **Modular Components:** Easily coordinate development for Navbar, Work, Services, and more.
- **Brutalist Aesthetic:** Controlled typographic hierarchy with "Instrument Serif" and "Inter" fonts.
- **Performance Optimized:** Small disk footprint and optimized animations (respects prefers-reduced-motion).

## 🛠️ Tech Stack
- **Framework:** Next.js 14 (App Router)
- **3D Engine:** React Three Fiber (@react-three/fiber, @react-three/drei)
- **Animation:** Framer Motion
- **Styling:** Tailwind CSS (v4)
- **Icons:** Lucide React

## 📦 Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
